# <a name="compiler-warning-c5045"></a>컴파일러 경고 C5045

> 컴파일러가/Qspectre 스위치가 지정 된 경우 메모리 로드를 위해 스펙터 완화를 삽입 합니다.

## <a name="remarks"></a>주의

경고 C5045를 사용 하면 [/Qspectre](../../build/reference/qspectre.md) 컴파일러 옵션이 지정 된 경우 코드에서 lfence와 같은 스펙터 완화를 삽입 하는 패턴을 확인할 수 있습니다. 이를 통해 보안 문제의 영향을 받는 코드 파일을 식별할 수 있습니다. 이 경고는 전적으로 정보를 제공 합니다. 즉, **/Qspectre** 스위치를 사용 하 여 다시 컴파일할 때까지 완화가 삽입 되지 않습니다. C5045의 기능은 **/Qspectre** 스위치에 독립적 이므로 동일한 컴파일에서 둘 다 사용할 수 있습니다.

이 경고는 Visual Studio 2017 버전 15.7에서 새로 발생 하며 기본적으로 해제 되어 있습니다. [/Wall](../../build/reference/compiler-option-warning-level.md) 를 사용 하 여 기본적으로 해제 된 모든 경고를 사용 하도록 설정 하거나 __/w__*n*__5038__ 을 사용 하 여 수준 *n* 경고로 C5045를 사용 하도록 설정 합니다. IDE에서 기본 경고 수준은 **/W3** 이 고 프로젝트 **속성 페이지** 대화 상자에서이 경고를 사용 하도록 설정할 수 있습니다. **구성 속성** > **C/C++**  > **명령줄** 을 열고 **추가 옵션** 상자에서 */w35045*을 추가한 다음 **확인**을 선택 합니다. 자세한 내용은 [기본적으로 해제 되어 있는 컴파일러 경고](../../preprocessor/compiler-warnings-that-are-off-by-default.md)를 참조 하세요. 컴파일러 버전 별로 경고를 사용 하지 않도록 설정 하는 방법에 대 한 자세한 내용은 컴파일러 [버전 별 컴파일러 경고](compiler-warnings-by-compiler-version.md)를 참조 하세요.

## <a name="example"></a>예제

다음 예제에서는 **/Wall** 또는 **/w35045** 및 **/W3** 옵션을 사용 하 여 Visual Studio 2017 버전 15.7에서 컴파일된 경우 경고 C5045를 발생 시킵니다.

```cpp
// C5045.cpp
// Compile with: cl /EHsc /W3 /w35045 C5045.cpp

int G, G1, G2;

__forceinline
int * bar(int **p, int i)
{
    return p[i];
}

__forceinline
void bar1(int ** p, int i)
{
    if (i < G1) {
        auto x = p[i]; // C5045: mitigation here
        G = *x;
    }
}

__forceinline
void foo(int * p)
{
    G = *p;
}

void baz(int ** p, int i)
{
    if (i < G1) {
        foo(bar(p, i + G2));
    }
    bar1(p, i);
}

int main() { }
```

경고를 사용 하는 경우 컴파일러 출력은 다음과 같습니다.

```Output
C:\Users\username\source\repos\C5045>cl /W3 /w35045 C5045.cpp
Microsoft (R) C/C++ Optimizing Compiler Version 19.14.26431 for x86
Copyright (C) Microsoft Corporation.  All rights reserved.

C5045.cpp
c:\users\username\source\repos\c5045\c5045.cpp(16) : warning C5045: Compiler will insert Spectre mitigation for memory load if /Qspectre switch specified
c:\users\username\source\repos\c5045\c5045.cpp(15) : note: index 'i' range checked by comparison on this line
c:\users\username\source\repos\c5045\c5045.cpp(17) : note: feeds memory load on this line
Microsoft (R) Incremental Linker Version 14.14.26431.0
Copyright (C) Microsoft Corporation.  All rights reserved.

/out:C5045.exe
C5045.obj
```

경고 메시지는 줄 16에 완화가 삽입 되었음을 보여 줍니다. 또한 줄 15의 인덱스 i가 줄 17의 메모리 로드를 피드 하므로 완화가 필요 합니다. 추론은 막대와 bar1에서 수행 되지만 완화는 줄 16에 배치 될 때 적용 됩니다.

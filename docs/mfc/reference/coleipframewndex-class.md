# <a name="coleipframewndex-class"></a>COleIPFrameWndEx 클래스

`COleIPFrameWndEx` 클래스는 MFC를 지원하는 OLE 컨테이너를 구현합니다. 애플리케이션에 대한 현재 위치의 프레임 창 클래스는 `COleIPFrameWndEx` COleIPFrameWnd [클래스에서 파생하는 대신](../../mfc/reference/coleipframewnd-class.md)클래스에서 파생해야 합니다.
더 자세한 내용은 Visual Studio 설치의 **VC\\atlmfc\\src\\mfc** 폴더에 있는 소스 코드를 참조하세요.

## <a name="syntax"></a>구문

```
class COleIPFrameWndEx : public COleIPFrameWnd
```

##  <a name="dockpaneleftof"></a>  COleIPFrameWndEx::DockPaneLeftOf

창을 다른 창의 왼쪽에 도킹합니다.

```
BOOL DockPaneLeftOf(
    CPane* pBar,
    CPane* pLeftOf);
```

### <a name="parameters"></a>매개 변수

*pBar*<br/>
진행 도킹할 창에 대 한 포인터입니다.

*pLeftOf*<br/>
진행 원본으로 사용 되는 창에 대 한 포인터입니다.

### <a name="return-value"></a>반환 값

작업이 성공 하면 TRUE를 반환 합니다. 그렇지 않으면 FALSE를 반환 합니다.

### <a name="remarks"></a>설명

여러 창 개체를 미리 정의 된 순서로 도킹 하려면이 메서드를 호출 합니다. 이 메서드는 *Pbar* 로 지정 된 창을 *pLeftOf*에 지정 된 창의 왼쪽에 도킹 합니다.
